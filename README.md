👔 ダメ社員マネージャー - アプリ説明書

ゲーム概要

「大事な要件がある日に限って体調不良になる」「提出物が出ていないと『今やろうと思ってました』と言う」

そんな世間で噂される「ダメ社員」たちをマネジメントし、胃に穴が開く前にプロジェクトを完遂させる、ブラック企業サバイバル・カードゲームです。

プレイヤーはプロジェクトマネージャーとなり、クセの強い社員（カード）たちの言い訳や行動を駆使して、納期までに進捗を100%にすることを目指します。

🎮 ゲームモード

1. 通常モード

手札の社員カードの内容（スキルやリスク）が見えている状態でプレイします。
戦略的に社員を選び、ストレスと進捗のバランスを取りながら進めるモードです。

2. ハードモード（暗闇のマネジメント）

**「履歴書在中」**という封筒しか見えない状態で、社員を採用（カード選択）しなければなりません。
めくるまで誰が来るか、どんな言い訳をするか分からない、運と勘が試される高難易度モードです。

納期が通常より短く設定されています（8日）。

ストレスの増加量が少し高くなります。

🚦 ルールと勝利条件

基本ルール

ターン制: カードを1枚選ぶと「1日」が経過します。

パラメータ:

進捗: 100%になるとゲームクリア。

ストレス: 社員の行動により上昇。100%になるとゲームオーバー。

納期: 0日になるとゲームオーバー。

終了条件

🎉 勝利: 納期内に「進捗」が100%に到達する。

💀 敗北:

「ストレス」が100%になり、胃に穴が開く。

進捗が100%未満のまま「納期」が尽きる（プロジェクト炎上）。

💡 攻略のヒント

ストレス管理が命: 進捗ばかり優先すると、あっという間にストレスで倒れます。「午後から帰ります」カードなどをうまく使い、ガス抜きをしましょう。

ギャンブル枠の活用: 「大事な日に体調不良」や「謎の自信」はハイリスク・ハイリターンです。納期の余裕があるうちに試すのが吉です。

ハードモードの心構え: 誰が来るか分からない恐怖を楽しみましょう。理不尽こそがこのゲームの醍醐味です。
